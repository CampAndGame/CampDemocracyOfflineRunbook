# Как запустить стенд Democracy

Простая инструкция. Нужен только установленный **Docker**.

---

## 1. Установи Docker (если ещё нет)
- Windows / macOS — **Docker Desktop** (docker.com → Download).
- Linux — **Docker Engine + Compose plugin**.
Проверка: открой терминал, набери `docker --version` — должна показаться версия.

## 2. Впиши пароль админки
Открой файл **`.env`** текстовым редактором (если его нет — он создастся на шаге 4, тогда впиши и запусти снова).
В строке `ADMIN_LOCAL_PASSWORD=` напиши свой пароль, например:
```
ADMIN_LOCAL_PASSWORD=stand2026
```
Сохрани файл. **Больше ничего в нём трогать не надо** — остальное подставится само.

## 3. Положи игру
Скачанный в онлайне файл игры (`bundle.zip`) положи в папку **`bundles/`**.
> Можно и без этого — игру можно загрузить позже прямо в админке.

## 4. Запусти
- Windows — двойной клик по **`start.bat`** (или в терминале `start.bat`).
- macOS / Linux — в терминале: `./start.sh`

Первый запуск — пара минут (грузятся образы). Дальше — секунды.

## 5. Открой в браузере
- **Админка (ведущий):** http://localhost/admin/ — вход по паролю из шага 2.
- **Плеер (команды):** http://localhost/ — команды заходят по коду из админки (вкладка «Команды»).

---

## Если что-то не так
- **Просит вписать пароль** — сделай шаг 2 и запусти снова.
- **Порт 80 занят** — открой `docker-compose.yml`, поменяй `"80:80"` на `"8080:80"`, тогда адреса будут `http://localhost:8080/...`.
- **Остановить стенд:** `docker compose down` (данные сохранятся).
- **Логи / что происходит:** `docker compose logs -f democracy-api`.
